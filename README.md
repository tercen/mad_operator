# MAD

##### Description

The `MAD operator` computes the Median Absolute Deviation (MAD) of the data points in a cell.

##### Usage

Input projection|.
---|---
`y-axis`        | numeric, measurement 

Output relations|.
---|---
`mad`        | median absolute deviation, per cell
